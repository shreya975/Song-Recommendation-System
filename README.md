# 🎵 AI Song Recommendation System

An AI-powered music recommendation application that suggests songs based on their audio features and musical characteristics. The system analyzes song attributes using Machine Learning to deliver personalized music recommendations through an interactive Streamlit web application.

# 📌 Project Overview

Finding songs that match your musical taste can be challenging with millions of tracks available on streaming platforms.

This project uses Machine Learning to recommend similar songs based on audio features such as danceability, energy, tempo, loudness, acousticness, and more. Users simply select a song, and the application instantly generates highly relevant recommendations.

The application combines recommendation algorithms with an elegant and interactive Streamlit interface.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/shreya975/Song-Recommendation-System.git
```

## Navigate into Project

```bash
cd Song-Recommendation-System
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Application

```bash
streamlit run app.py
```
